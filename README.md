## Install
```sh
npm install node-red-contrib-bluelinky
```

## Documentation
Checkout out the docs [bluelinky-docs](https://hacksore.github.io/bluelinky-docs/) for more info.

## Supported Features
- Lock
- Unlock
- Start (with climate control)
- Stop
- Status
